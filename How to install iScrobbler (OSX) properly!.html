How to install iScrobbler (OSX) properly!

This short instruction should help all OSX users to install iScrobbler. This will also make sure that iScrobbler starts up every time you switch on your computer, so you don't have to start it up yourself every time.

  • Step 1: download and unstuff iScrobbler.

  • Step 2: copy iScrobbler to your applications folder on your system hard disk.

  • Step 3: make iScrobbler a statup item; go to the 'apple' menu, find the 'system preferences' and click on 'account'. then go to 'startup items' and click the add '+' button. select iScrobbler. that's it! now iScrobbler is on automatically every time you boot up your computer.
